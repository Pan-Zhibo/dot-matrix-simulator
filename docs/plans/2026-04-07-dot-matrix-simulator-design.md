# 点阵屏模拟器 — 设计文档

> 版本：V1.0
> 日期：2026-04-07
> 负责人：潘志博

---

## 1. 项目概述

**用途**：调试用点阵屏模拟器，用于预览文字在MAX7219点阵屏上的显示效果。

**输出**：单个HTML文件，可离线使用，可部署到GitHub Pages。

---

## 2. 功能规格

### 2.1 布局配置

| 功能 | 说明 |
|------|------|
| 竖排数量 | 输入框，范围1~10，默认3 |
| 横排数量 | 输入框，范围1~10，默认4 |
| 点阵单元 | 每单元8×8像素组 |
| 实时预览 | 配置改变后立即刷新显示区域 |

### 2.2 文字显示

| 功能 | 说明 |
|------|------|
| 文字输入 | 文本框，支持任意字符 |
| 字体大小 | 滑块调节，范围8px~24px |
| 显示模式 | 滚动字幕 / 静态显示，可切换 |
| 滚动速度 | 数字输入框，单位ms/步，默认50ms |
| 滚动方向 | 从右往左 |

### 2.3 颜色控制

| 功能 | 说明 |
|------|------|
| 每组颜色 | 每个8×8单元独立设置颜色 |
| 预设色板 | 8个快捷颜色：红、绿、蓝、黄、青、粉、橙、白 |
| 颜色选择器 | 完整调色盘，自由调配 |
| 界面配色 | 深色背景，还原真实显示效果 |

### 2.4 其他功能

| 功能 | 说明 |
|------|------|
| 亮度调节 | 整体亮度滑块，范围20%~100% |
| 清屏 | 按钮，点击后所有LED熄灭 |
| 全亮测试 | 按钮，点击后所有LED亮起 |

## 3. 技术方案

### 3.1 技术栈

- **纯HTML + CSS + JavaScript**，无任何外部依赖
- **Canvas 2D** 绘制点阵显示区域
- **Google Fonts**：Roboto Mono（等宽字体，显示更整齐）
- **响应式设计**：适配手机和桌面浏览器

### 3.2 字体渲染

- 使用 `CanvasRenderingContext2D.fillText()` 绘制文字
- 文字裁剪：超出显示区域的文字在滚动模式下逐步进入视野
- LED效果：通过 `shadowBlur` 实现发光效果，增强真实感

### 3.3 文件结构

```
dot-matrix-simulator.html  ← 唯一文件，全部内联
```
